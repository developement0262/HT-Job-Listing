== Description ==
This plugin is used to show job listing where user can go to company and check that company available jobs and apply it. 

== Installation ==
1. Upload `ht-job-listing.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Place `[company_job_listing]` shortcode in your widget of your group page.
